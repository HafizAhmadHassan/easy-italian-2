# 📘 Super Easy Italian – Study Sheet

## 🎯 1. Core Pattern: **STO + GERUND** (present continuous)

👉 Used for actions happening **right now**

### 🔑 Structure:

**stare (conjugated) + verb (-ando / -endo)**

| Italian              | English            |
| -------------------- | ------------------ |
| sto facendo il caffè | I am making coffee |
| sto lavorando        | I am working       |
| sto leggendo         | I am reading       |
| sto bevendo          | I am drinking      |

---

## 🎯 2. Core Pattern: **AVERE expressions**

👉 Italian uses **“to have”**, not “to be”

### 🔥 Most important expressions:

| Italian        | Literal       | Real English |
| -------------- | ------------- | ------------ |
| ho fame        | I have hunger | I’m hungry   |
| ho sete        | I have thirst | I’m thirsty  |
| ho caldo       | I have heat   | I’m hot      |
| ho freddo      | I have cold   | I’m cold     |
| ho sonno       | I have sleep  | I’m sleepy   |
| ho voglia di…  | I have desire | I feel like… |
| ho bisogno di… | I have need   | I need…      |

---

## ⚠️ Important mistake to avoid

❌ *sono caldo* → WRONG (can sound inappropriate)
✅ *ho caldo* → correct

❌ *sono 37 anni*
✅ *ho 37 anni*

---

## 🎯 3. Present of “avere”

| Person  | Italian | English      |
| ------- | ------- | ------------ |
| io      | ho      | I have       |
| tu      | hai     | you have     |
| lui/lei | ha      | he/she has   |
| noi     | abbiamo | we have      |
| voi     | avete   | you all have |
| loro    | hanno   | they have    |

👉 The **“h” is silent**

---

## 🎯 4. Past (Imperfetto) – ongoing past

👉 Used for things that **were true for some time**

| Italian        | English          |
| -------------- | ---------------- |
| avevo fame     | I was hungry     |
| avevi sete     | you were thirsty |
| aveva caldo    | he/she was hot   |
| avevamo voglia | we felt like     |
| avevano freddo | they were cold   |

---

## 🎯 5. Useful Question Pattern

### 🗣️ “What are you doing?”

👉 **Cosa stai facendo?**

Breakdown:

* cosa = what
* stai = you are
* facendo = doing

Think like this:

👉 English: “I AM hungry”
👉 Italian: **“I HAVE hunger”**

So anytime it’s a **feeling or physical state → use AVERE**
